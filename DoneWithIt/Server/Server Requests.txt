General text request format:
Delimiter list: "~|`"

Message to Server format:
{request_number}~|`{userID/email}~|`{case specific data}

Server response format:
{request_number}~|`{relevant data}

1. Check User Exists
    Format: 1~|`email
    > email

    < true/false

2. Add new User
    Format: 2~|`email~|`displayName~|`
    > email
    > displayName
    > currentDate

    < userID

3. Get existing user data
    Format: 3~|`email
    > email

    < userID
    < profile picture
    < outfit images and associated file paths


4. Update display name
    Format: 4~|`userID~|`displayName
    > userID
    > display name

    < succes/fail


5. Update profile pic
    Format: 5~|`userID~|`{need to figure out what to do with pics for the moment}
    > userID
    > new picture

    < success/failure


6. Remove all user data 
    Format: 6~|`userID
    > userID

    < success/fail


7. Add new outfit to database under user
    Format: 7~|`userID~|`{outfit image idk what to do so far}~|`outfitName~|`outfitDescription
    > user Id
    > outfit image
    > outfit name
    > outfit description
    > creationDate

    < success/failure
    < outfitID


8. Delete outfit from database
    Format: 8~|`userID~|`outfitID
    > userID
    > outfitID

    < success/failure
    

9. Edit outfit information in database
    Format: 9~|`userID~|`outfitID~|`{outfit Image TODO}~|`outfitName~|`outfitDescription
    > userID
    > outfitID
    > outfit image
    > outfit name
    > outfit description

    < success/failure
    < outfitID


10. Give/receive notification token
    Format: 10~|`userID~|`token/NULL
    > userID
    > token || NULL

    < success/failure || token
